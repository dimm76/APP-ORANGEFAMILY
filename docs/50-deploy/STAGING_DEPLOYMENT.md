# OrangeFamily Staging Deployment

## Estado

OrangeFamily todavía no dispone de un entorno de staging configurado.

Este documento se completará cuando se definan y validen:

- dominio del frontend;
- dominio de la API;
- VPS;
- rutas de despliegue;
- servicio Node;
- puerto interno;
- base de datos PostgreSQL;
- variables de entorno;
- flujo de despliegue;
- validaciones posteriores al despliegue.

No deben copiarse configuraciones, credenciales, rutas, servicios ni bases de datos de OrangeTraining u OrangeDesk sin una decisión explícita.
